MySQL Storage

Every event is immediately persisted to MySQL inside the user_logs table. The schema includes: 

CREATE TABLE user_logs (
  id         INT AUTO_INCREMENT PRIMARY KEY,
  username   VARCHAR(100),
  action     VARCHAR(100),
  source_ip  VARCHAR(50),
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

Records can be browsed on the MySQL Logs page with filtering and pagination.

External HTTP Endpoint

Simultaneously with the MySQL write, the Flask backend sends an HTTP POST request to a configurable external URL with a JSON body: 

{
  "username":  "alice_smith",
  "action":    "login",
  "source":    "web_app",
  "source_ip": "192.168.1.10",
  "timestamp": "2025-01-15T08:23:11Z"
}

The endpoint URL is set in app.py via EXTERNAL_LOG_ENDPOINT and can also be changed at runtime on the Event Generator page.

API Reference
